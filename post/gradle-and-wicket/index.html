Gradle and Wicket

Apr 30, 2011   #build  #gradle  #wicket 

To make sure Gradle packs your *.html files into your WAR file when developing a Wicket application, include your Java directory as a resource directory in your build.gradle:

{% codeblock lang:java%} sourceSets { main { java { srcDir ‘src/main/java’ } resources { srcDir ‘src/main/resources’ srcDir ‘src/main/java’ } } } ```
